# Contexto

Produto de gestao financeira com interface principal por voz. O usuario descreve gastos, recebimentos, compromissos e pagamentos em linguagem natural, enquanto o backend organiza os dados e a interface principal inicial roda como PWA para facilitar testes em iPhone e Android.

Nesta fase, o objetivo e montar a base tecnica correta para crescimento por etapas, mantendo o backend como orquestrador principal e o toque como fallback da experiencia.
